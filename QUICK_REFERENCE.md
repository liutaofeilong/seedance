## 视频参数配置

### 视频比例
```
21:9  - 电影超宽屏（适合影院风格）
16:9  - 标准宽屏（YouTube、电视）
4:3   - 传统比例（复古风格）
1:1   - 正方形（Instagram）
3:4   - 竖屏（手机浏览）
9:16  - 手机竖屏（TikTok、Reels）
```

### 分辨率
```
480p  - 标清（快速生成，文件小）
720p  - 高清（推荐，平衡质量和速度）
1080p - 全高清（最佳质量，耗时长）
```

### 时长
```
3-10秒 - 滑块选择
推荐：5秒（默认）
```

### 音频选项
```
🔊 包含声音 - 自动生成与画面匹配的音效和背景音
🔇 无声视频 - 纯视觉内容
```

### 附加选项
```
💧 水印
  - 开启：显示平台水印
  - 关闭：无水印（推荐）

📷 镜头模式
  - 固定镜头：静态拍摄
  - 运动镜头：动态镜头运动（推荐）
```

## 提示词技巧

### 文生视频
```
✅ 好的提示词：
"无人机以极快速度穿越复杂障碍或自然奇观，带来沉浸式飞行体验"

❌ 避免：
"视频" "生成" 等元描述词
```

### 图生视频
```
✅ 单张图片（首帧）：
"镜头缓慢向前推进，展现画面中的细节"

✅ 两张图片（首尾帧）：
"从第一张图片平滑过渡到第二张图片，保持流畅的运动"
```

## API 参数映射

### 在提示词中的格式
```
--duration 5          # 时长5秒
--ratio 16:9          # 宽高比16:9
--camerafixed false   # 运动镜头
--watermark false     # 无水印
```

### 完整示例
```
提示词：无人机穿越峡谷 --duration 5 --ratio 16:9 --camerafixed false --watermark false
```

## 常见问题

### Q: 哪个模型最好？
A: Doubao-Seedance-1.5-pro 是当前推荐的主力模型，支持音频生成。

### Q: 视频生成需要多久？
A: 通常 1-3 分钟，取决于队列和参数复杂度。

### Q: 可以生成多长的视频？
A: 当前支持 3-10 秒，推荐 5 秒。

### Q: 音频是如何生成的？
A: 1.5-pro 模型会根据视觉内容和提示词自动生成匹配的音效和背景音。

### Q: 首尾帧模式有什么用？
A: 可以精确控制视频的开始和结束画面，适合制作特定的过渡效果。

## 故障排除

### 任务一直在 queued
- 正常现象，等待队列处理
- 通常 30秒内会变为 running

### 任务失败
- 检查提示词是否包含敏感内容
- 检查图片格式和大小
- 尝试简化提示词

### 网络超时
- 系统会自动重试
- 不影响任务执行
- 继续等待即可

### 视频质量不理想
- 尝试更详细的提示词
- 使用更高分辨率
- 尝试不同的模型
